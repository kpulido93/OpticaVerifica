# Optima Veriifica

Este repositorio corresponde a **Optima Veriifica**.

## Configuración de entorno

Usa el archivo de ejemplo para variables de entorno:

```bash
cp .env.example .env
```

La zona horaria por defecto del ejemplo es:

- `TZ=America/Santo_Domingo`
